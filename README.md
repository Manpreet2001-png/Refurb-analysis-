# Blackbelt Pricing App

📱 **Description:**  
This is a Streamlit app that calculates pricing for failed devices by analyzing Blackbelt test reports. Perfect for technicians, repair shops, or anyone tired of doing this manually.  

🛠 **Features:**  
- Upload Blackbelt test reports (Excel/CSV)  
- Automatically detect failed devices  
- Calculate pricing for each failed part  
- Clean, interactive dashboard with Streamlit  

💻 **How to Run Locally:**  
1. Clone this repo:  
```bash
git clone https://github.com/YOUR_USERNAME/blackbelt-pricing-app.git
pip install -r requirements.txt
